Random Project
=======

Este repositorio contiene el proyecto Random, que contiene un fichero random.js que al ejecutarse, muestra por consola si un nÃºmero aleatorio es mayor o menor que 0.5

Ejemplo de uso:

```node
> nodejs random
< 0.8820516963023692 MAYOR que 0,5

> nodejs random
< 0.15139998495578766 MENOR que 0,5
```
